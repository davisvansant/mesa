###### | mesa

A CLI based tool to build, test, and run `AWS Lambda` code

```
mesa 0.1.0
| mesa - an isolated place

USAGE:
    mesa [SUBCOMMAND]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

SUBCOMMANDS:
    build     build and create your mesa
    erode     cleanup and remove your mesa
    form      form (finalize and release) your mesa
    help      Prints this message or the help of the given subcommand(s)
    survey    survey (prepare and evaluate) to form your mesa
    view      view (test) your mesa
```
